## Prerequisites

- [Fauna](https://fauna.com/)

## Getting Started

Rename `.env.example` to `.env.local` and values to the variables below.

| KEY | VALUE |
| --- | ----- |
| `API_ROOT` | Domain name to prefix shortened IDs with. (Example: https://f-st.ga) |
| `DB_KEY` | Fauna DB API key.
