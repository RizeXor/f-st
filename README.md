## Getting Started

**Prerequisites**

- [Fauna](https://fauna.com/)

## Configuration

For configuration edit `.env.local` or use environment variables.

- `API_ROOT` Backend API Base URL (eg:- https://f-st.ga/jpIo4W)
- `FAUNA_KEY` Fauna API KEY
